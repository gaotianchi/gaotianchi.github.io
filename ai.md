## 理念

这个站点是人类写作的空间。所有文章均为我一个人——高天驰——独立构思、撰写与修订。文字中的感受、困惑、反思与洞见，皆出自有限而真实的人生经验。

## 使用范围

AI（主要为 Claude）在这个站点中的参与仅局限于以下方面：

**技术开发**
- 站点结构改版与维护（Jekyll 模板、样式、配置）
- 文章元数据批量处理（如归档操作）
- Git 操作辅助

**写作辅助**（极少数情况）
- 英文或中文表达的局部润色建议
- 复杂句式的可读性调整

**从未涉及的领域**
- 文章立意、观点与结论——全部源于我个人
- 文章的整体结构与叙事——完全由我掌控
- 任何署名文章的主体内容——逐字逐句皆为我所写

## 原则

{% capture quote %}
人的思考不可外包。AI 可以打磨工具，但不能替代握工具的手。
{% endcapture %}

我的底线是：**技术可以参与表达的形式，但不能僭越表达的主体。** 这个站点之所以存在，是因为写作本身即是目的——理解自己、厘清思绪、对抗遗忘。如果把这一切交给机器，那么这个站点便失去了存在的意义。
