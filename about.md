## 我

人类的一员。目前活跃在中国。

### 三个支柱

**健康** / **自我** / **爱**

它们支撑着我。如果没有它们，我将无法坦然面对各种形式的离别——无论是空间上的、生与死的、还是情感上的；也无法像现在这样，隐入并享受孤独的时光。

我总是感谢自己，这不是没有道理。

## 博客

博客一种由个人或小团队维护、以时间倒序（最新的内容在最上方）排列、定期更新的“数字化个人日志”或“在线个人出版空间”。

### 历史沿革

- 2022.09.03 / **初创**
	
	初次建站，名为“悲伤的斯蒂梵冈”。 
	
- 2022 — 2025 / **折腾**
	
	期间频繁更换域名、网站名称和建站工具。我曾用过 WordPress、Blogger，甚至用 Flask 框架自己从零开发过博客网站。但最终都不了了之——我将过多的精力投入到了工具的折腾上，却轻视了内容本身。 
	
- 2025.05.05 / **回归**
	
	再次建站，决定用自己的名字命名。它不再混杂，而是极其纯粹：专注于探究自我，通过写作疗愈内心，打开心结。

### 当前架构 

通过 Jekyll 搭建，部署于 GitHub Pages，主题采用 Klise。 

受限于 GitHub 部署，国内访问速度非常慢。但我暂时不打算把精力花在技术优化上。
